This tool will open a list of urls into a separate minimized window. Each tab will be opened once the prior tab has been loaded fully. Once all tabs have been opened, this tool's tab will close. As opposed to other mass-opening tools, this tool may be useful for: reducing the spike in bandwidth usage, avoiding 503 errors from making too many requests to the same website too quickly, etc. Click 'Open All' without any urls to see an example.
